# 消防商机搜索器

搜索并返回消防相关的商业机会信息，包括招标公告、采购项目等。

## 使用方法

直接执行即可搜索消防商机，无需任何参数。

## 参数说明

| 参数 | 说明 |
|------|------|
| `query` | 搜索关键词（可选） |
| `limit` | 返回结果数量（默认5条） |
