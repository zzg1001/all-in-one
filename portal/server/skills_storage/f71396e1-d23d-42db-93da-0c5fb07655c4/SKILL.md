# 财务分析报告生成技能

将Excel财务数据转换为专业的PDF分析报告，风格参考IMS全局效率月报。

## 报告结构

生成的PDF报告包含以下7大模块：

### 1. Executive Summary（执行摘要）
顶部KPI卡片展示4个核心指标：
- 营业收入（含预算完成率）
- 毛利率（含与预算差异pp）
- 净利润（含预算完成率）
- 费用率（含与预算差异pp）

### 2. 关键财务指标分析
表格展示核心经营指标：
| 指标 | 本期实际 | 本期预算 | 上期实际 | 单位 | 完成率 | 环比 |
|------|----------|----------|----------|------|--------|------|
| 营业收入 | | | | 万元 | | |
| 营业成本 | | | | 万元 | | |
| 毛利润 | | | | 万元 | | |
| 毛利率 | | | | % | | |
| 营业利润 | | | | 万元 | | |
| 净利润 | | | | 万元 | | |
| 净利率 | | | | % | | |

### 3. 收入趋势分析
- 柱状图：月度收入预算vs实际对比（12个月）
- 差异标注：每个月的差异值，正值绿色，负值红色
- 文字分析要点

### 4. 产品线盈利分析
- 饼图：各产品线收入占比
- 条形图：各产品线毛利率对比
- 表格：产品线明细（收入、成本、毛利、毛利率、占比）

### 5. 部门费用执行分析
- 柱状图：各部门预算vs实际对比
- 完成率标注
- 表格：部门费用明细（预算、实际、差异、完成率、状态）
- 状态判断：>105%超支，100-105%关注，<100%正常

### 6. 费用明细分析
表格展示各费用类别：
- 人工成本、办公费用、差旅费、营销推广、招待费、培训费、其他
- 本月实际/预算、差异、累计实际、年度预算、执行率

### 7. 现金流量分析
- 折线图：12个月现金流趋势
- 三条线：经营性现金流、投资性现金流、筹资性现金流
- 零线参考
- 文字分析要点

### 8. 风险提示与管理建议
表格展示风险事项：
| 风险等级 | 风险事项 | 影响程度 | 建议措施 |
|----------|----------|----------|----------|
| 高/中/低 | | | |

颜色编码：高风险红色背景，中风险黄色，低风险绿色

## 输入数据格式

Excel文件应包含以下工作表（可选，根据实际数据情况）：

### Sheet 1: 月度数据
| 月份 | 收入预算(万元) | 收入实际(万元) | 成本预算(万元) | 成本实际(万元) | 费用预算(万元) | 费用实际(万元) |
|------|----------------|----------------|----------------|----------------|----------------|----------------|

### Sheet 2: 部门费用
| 部门 | 预算(万元) | 实际(万元) |
|------|------------|------------|

### Sheet 3: 产品线分析
| 产品线 | 收入(万元) | 成本(万元) |
|--------|------------|------------|

### Sheet 4: 关键指标
| 指标名称 | 本期实际 | 本期预算 | 上期实际 | 单位 |
|----------|----------|----------|----------|------|

### Sheet 5: 费用明细
| 费用类别 | 本月实际(万元) | 本月预算(万元) | 累计实际(万元) | 年度预算(万元) |
|----------|----------------|----------------|----------------|----------------|

## 使用方式

### 方式一：使用现有Excel数据
```
用户提供Excel文件路径，运行脚本生成PDF报告
```

### 方式二：生成示例数据
如果用户没有现成数据，可以使用脚本生成示例数据：
```bash
python scripts/generate_finance_report.py --generate-sample
```

### 方式三：完整流程
```bash
python scripts/generate_finance_report.py --input <excel_path> --output <pdf_path>
```

## 执行步骤

1. **检查用户输入**
   - 如果用户提供了Excel文件，使用该文件
   - 如果没有，询问是否需要生成示例数据

2. **运行报告生成脚本**
   ```bash
   python <skill_path>/scripts/generate_finance_report.py --input <excel_path> --output <output_dir>
   ```

3. **返回结果**
   - 告知用户PDF报告生成路径
   - 简要说明报告包含的内容

## 设计规范

### 颜色体系
- 主色调：#3498DB（蓝色）
- 正向指标：#27AE60（绿色）
- 负向/风险：#E74C3C（红色）
- 关注/警告：#F39C12（橙色）
- 背景：#F8F9FA（浅灰）
- 文字：#2C3E50（深灰）

### 表格样式
- 表头：彩色背景+白色文字
- 斑马纹：交替行背景色
- 边框：0.5pt浅灰色

### 图表样式
- 标题加粗居中
- 图例位于右上或下方
- 网格线半透明
- 数据标签清晰
