# 人力资源分析报告生成技能

将Excel人力资源数据转换为专业的PDF分析报告。

## 报告结构

生成的PDF报告包含以下7大模块：

### 1. Executive Summary（执行摘要）
顶部KPI卡片展示4个核心指标：
- 员工总数（含环比变化）
- 本月离职率（含与目标差异）
- 招聘完成率（含目标对比）
- 人均薪酬（含环比变化）

### 2. 人员结构分析
- 饼图：各部门人员占比
- 柱状图：各部门人数分布
- 表格：部门明细（人数、占比、环比变化）

### 3. 人员流动分析
- 柱状图：月度入职/离职趋势（12个月）
- 表格：各部门离职情况（离职人数、离职率、状态）
- 离职原因分析饼图

### 4. 招聘分析
- 柱状图：招聘计划vs实际完成
- 表格：各渠道招聘效果（渠道、简历数、面试数、录用数、转化率）
- 招聘成本分析

### 5. 薪酬分析
- 柱状图：各部门薪酬总额对比
- 表格：薪酬明细（部门、薪酬总额、人均薪酬、环比）
- 薪酬结构饼图（基本工资、绩效、补贴、其他）

### 6. 绩效分析
- 饼图：绩效等级分布（A/B/C/D）
- 柱状图：各部门平均绩效得分
- 表格：绩效明细

### 7. 风险提示与管理建议
表格展示HR风险事项：
| 风险等级 | 风险事项 | 影响程度 | 建议措施 |
|----------|----------|----------|----------|
| 高/中/低 | | | |

## 输入数据格式

Excel文件应包含以下工作表：

### Sheet 1: 员工花名册
| 工号 | 姓名 | 部门 | 岗位 | 入职日期 | 学历 | 性别 | 年龄 | 基本工资 |
|------|------|------|------|----------|------|------|------|----------|

### Sheet 2: 月度人员变动
| 月份 | 入职人数 | 离职人数 | 期初人数 | 期末人数 |
|------|----------|----------|----------|----------|

### Sheet 3: 部门统计
| 部门 | 编制人数 | 实际人数 | 离职人数 | 本月入职 | 本月离职 |
|------|----------|----------|----------|----------|----------|

### Sheet 4: 招聘数据
| 渠道 | 简历数 | 面试数 | 录用数 | 招聘成本(元) |
|------|--------|--------|--------|--------------|

### Sheet 5: 薪酬数据
| 部门 | 基本工资(万) | 绩效工资(万) | 补贴(万) | 其他(万) |
|------|--------------|--------------|----------|----------|

### Sheet 6: 绩效数据
| 部门 | A级人数 | B级人数 | C级人数 | D级人数 | 平均得分 |
|------|---------|---------|---------|---------|----------|

## 使用方式

### 方式一：使用现有Excel数据
```bash
python scripts/generate_hr_report.py --input <excel_path> --output <output_dir>
```

### 方式二：生成示例数据
```bash
python scripts/generate_hr_report.py --generate-sample --output <output_dir>
```

## 执行步骤

1. **检查用户输入**
   - 如果用户提供了Excel文件，使用该文件
   - 如果没有，询问是否需要生成示例数据

2. **运行报告生成脚本**
   ```bash
   python <skill_path>/scripts/generate_hr_report.py --input <excel_path> --output <output_dir>
   ```

3. **返回结果**
   - 告知用户PDF报告生成路径
   - 简要说明报告包含的内容

## 设计规范

### 颜色体系
- 主色调：#2E86AB（蓝色）
- 正向指标：#28A745（绿色）
- 负向/风险：#DC3545（红色）
- 关注/警告：#FFC107（黄色）
- 紫色：#6F42C1
- 背景：#F8F9FA
- 文字：#212529

### HR指标说明
- 离职率 = 离职人数 / 期初人数 × 100%
- 招聘完成率 = 实际录用 / 招聘计划 × 100%
- 人均薪酬 = 薪酬总额 / 员工人数
